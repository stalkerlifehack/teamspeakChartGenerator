# Requirements/wymagania
  - php7.2 or higher
  - php7.2-gd
  - screen

# Instalation/Instalacja
```sh
$ cd /home
$ wget https://github.com/stalkerlifehack/teamspeakChartGenerator
$ unzip master.zip
$ cd teamspeakChartGenerator
$ chmod 0775 run
```

# Commands/Komendy
```sh
$ ./run start
$ ./run stop
```
